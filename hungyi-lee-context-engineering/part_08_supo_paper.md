# Part 08: 強化學習優化摘要 (SUPO)

除了 ACON 之外，還有一篇非常厲害的論文叫做 **SUPO (Summarization augmented Policy Optimization)**
SUPO 解決的是一個更難的問題：當我們不知道「正確的摘要」長什麼樣子時，該怎麼辦？
在很多複雜任務中，我們很難定義哪些資訊是絕對重要的
SUPO 的做法是利用**強化學習 (Reinforcement Learning)**

具體的流程是這樣的：
1. 讓模型嘗試產生不同版本的摘要
2. 讓 Agent 根據這些摘要去解題
3. 如果 Agent 最後把題目解對了，就給該摘要正向獎勵
4. 如果解錯了，就給負向獎勵

透過這種方式，模型會慢慢學會：
「原來在這種情境下，保留這幾行程式碼比保留那一串對話更重要」
這不僅提升了模型的摘要能力
更讓 Agent 學會了如何在資訊有限的情況下執行任務
這就是 Context Engineering 的最高境界
讓模型具備「洞察重點」的能力，而這種能力是可以透過強化學習不斷進化的
這也讓 AI Agent 從一個「聽命行事」的工具，變成了一個能「自我優化」的智慧體
